# Wiki — структурное знание проекта {{PROJECT_NAME}}

Здесь живёт «что есть в коде» — данные, маршруты, архитектура. Обновляй
**вручную** при каждом значимом изменении (новая миграция / новый endpoint /
новый сервис). Если ответ есть здесь — Claude обязан искать сначала тут,
не парсить код заново.

Разница с `docs/`:
- **`wiki/`** = структура (производное от кода, snapshot текущего состояния).
- **`docs/`** = процесс (первичные нарративы: сессии, audits, решения).

## Карта

- [Модель данных](data-model.md) — схема БД, ER, ключевые ассоциации
- [API / маршруты](api.md) — endpoints, контракты, формат ответов
- [Архитектура](architecture.md) — компоненты, потоки данных, внешние зависимости
- [Пробелы](gaps.md) — что ещё не задокументировано структурно
- [Журнал изменений](log.md) — кто что обновил в wiki и когда
- [Шаблон страницы](TEMPLATE.md) — frontmatter + TL;DR

Подпапки растут вместе с кодом: `models/`, `controllers/`, `services/`.

## Правила работы

1. Wiki **не source of truth** — это удобная сводка. Source = код + миграции + ADR.
2. **Обновляй wiki в том же PR**, что меняет код. Если PR добавляет endpoint —
   обнови `api.md` в этом же PR. Reviewer проверяет это.
3. **НЕ автогенерируй wiki через git-хуки** на текущем этапе. Идея заманчива
   но (а) дрейф между авто-вики и кодом тих,
   (б) стоит $0.05-0.15 на коммит, (в) auto-generated wiki не может быть
   source of truth. Сначала ведём вручную, потом — когда увидим что
   устаревает быстрее всего — точечно автоматизируем для тех частей.
4. **Семантический поиск (QMD и аналоги)** — не нужен пока `wiki/` <50 файлов.
   Grep справится. Возврат к вопросу — когда количество файлов перейдёт порог.

## Каждая страница
YAML frontmatter (`title`, `type`, `source file path`, `created/updated date`,
`tags`, `confidence level`) + однострочный TL;DR + тело. Шаблон — `TEMPLATE.md`.

## Бесплатная автоматизация (единственная)
SessionStart hook (`.claude/hooks/session-start.sh`) при старте сессии выводит
шапку `index.md` + хвост `log.md`. Это только подсказка «загляни в wiki», не
авто-генерация — ничего не стоит и не дрейфует.
